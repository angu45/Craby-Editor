Introduction to Java

Java is a high-level, class-based, object-oriented programming language designed to have as few implementation dependencies as possible. Its famous slogan is: "Write Once, Run Anywhere" (WORA).

1. Key Features of Java

  • Object-Oriented: Everything in Java is an Object.
  • Platform Independent: Java code is compiled into bytecode, which runs on any device with a JVM.
  • Secure: Java provides a secure environment by eliminating pointers and running code inside a sandbox.
  • Robust: Strong memory management and exception handling make Java highly reliable.

2. The Java Ecosystem (JDK, JRE, JVM)

To run Java, you need to understand three core components:

Component Full Form Role
JVM Java Virtual Machine Executes the bytecode on the specific OS.
JRE Java Runtime Environment Contains JVM + Libraries to run Java apps.
JDK Java Development Kit Contains JRE + Tools (like javac) to develop Java apps.

3. Your First Java Program

Every Java program must have a class and a main method. Here is the standard "Hello World" example:

public class Main { public static void main(String[] args) { // This is a comment System.out.println("Hello, Welcome to Craby Editor!"); } }
Breakdown of the code:
  • public class Main: Defines a class named Main.
  • static void main: The entry point of any Java program.
  • System.out.println: Used to print text to the console.

4. How Java Works

1. You write source code in a .java file.
2. The compiler (javac) converts it into .class (bytecode) files.
3. The JVM translates bytecode into machine code that the computer understands.
