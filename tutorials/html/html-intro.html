Introduction to HTML 🌐

What is HTML?

HTML mhanje HyperText Markup Language. Hi language web pages cha structure banavnyasathi vaparli jate. Browser (Chrome, Firefox) ya HTML code la vachun tumhala ek sundar website dakhavto.

HTML shikne khup sopa ahe. Jar tumhala ek paragraph lihaycha asel kiwa ek image takaychi asel, tar tumhala fakt tya sathi aslele Tags mahit pahije.

HTML Structure Breakdown

Ek standard HTML page madhe khali dilele components hamesha astat:

  • <!DOCTYPE html>: He browser la sangte ki hi HTML5 file ahe.
  • <html>: Ha main container ahe, yachya aat sagle tags yetat.
  • <head>: Yat title, CSS links, ani Meta tags asat je page var disat nahit pan garjeche astat.
  • <body>: Yatla sagle content (Text, Images, Buttons) user la website var distat.

Let's Write Some Code!

Sample: Basic Page

<!DOCTYPE html>
<html>
<head>
  <title>My First Page</title>
</head>
<body>

  <h1>Welcome to Craby Editor</h1>
  <p>Start your coding journey here.</p>

</body>
</html>

Why is HTML Important?

Internet varil pratyek website, mag ti Google aso ki Facebook, HTML shivay banu shakat nahi. He web development che Foundational Pillar ahe. HTML shikle ki tumhi CSS ani JavaScript sahaj shikhu shakta.
