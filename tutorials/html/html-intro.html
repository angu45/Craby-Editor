Introduction to HTML 🌐

What is HTML?

HTML stands for HyperText Markup Language. This language is used to create the structure of web pages. Browsers (like Chrome or Firefox) read this HTML code and display it as a beautiful website.

Learning HTML is very easy. If you want to write a paragraph or insert an image, you just need to know the specific Tags used for those elements.

HTML Structure Breakdown

A standard HTML page always includes the following components:

  • <!DOCTYPE html>: This tells the browser that this is an HTML5 file.
  • <html>: This is the main container that holds all other tags.
  • <head>: This contains metadata, the title, and CSS links—essentials that don't appear on the page but are necessary for it to work.
  • <body>: All the visible content (Text, Images, Buttons) that users see on the website goes here.

Let's Write Some Code!

Sample: Basic Page

<!DOCTYPE html>
<html>
<head>
  <title>My First Page</title>
</head>
<body>

  <h1>Welcome to Craby Editor</h1>
  <p>Start your coding journey here.</p>

</body>
</html>

Why is HTML Important?

Every website on the internet, whether it's Google or Facebook, cannot be built without HTML. It is the Foundational Pillar of web development. Once you learn HTML, learning CSS and JavaScript becomes much easier.
