HTML Tags & Elements: The Deep Dive

HTML madhe pratyek tag cha ek vishisht 'Role' asto. Fakt text dakhvne mhanje HTML nahi, tar to text 'kaay' ahe he browser la samjavne mhanje HTML.

1. Text Formatting Tags

Text la bold, italic kiwa underline karnyathi he tags vaprtat. Pan he fakt styling sathi nahit, tar he content cha mahatva suddha dakhvtat.

  • <strong>: Text Bold karto (Important content sathi).
  • <em>: Text Italic karto (Emphasis sathi).
  • <mark>: Text Highlight karnyathi vaprtat.
  • <del>: Strike-through text sathi (Deleted content).
<p>He khup <strong>mahatvache</strong> ahe.</p>
<p>Aaj <mark>Offer</mark> suru ahe!</p>

2. Semantic Tags (Meaningful Structure)

Modern HTML5 madhe sagle <div> vaprun chalat nahi. Browser ani Google la website cha structure samjava mhanje 'Semantic Tags' garjeche ahet.

  • <header>: Website cha top part.
  • <nav>: Navigation links sathi container.
  • <main>: Page cha main unique content.
  • <footer>: Page cha shevatcha bhag (Copyright info).
  • <section>: Ek tharaavik group of content.

3. Links & Attributes

Attributes tags la 'Extra Information' detat. Jashe ki link kuthli ahe kiwa photo kiti motha pahije.

<!-- target="_blank" navin tab madhe link open karto -->
<a href="https://craby-editor.vercel.app" target="_blank">Go to Home</a>

<!-- title attribute hover kelyavar mahiti deto -->
<p title="I am a tooltip">Hover over me!</p>

4. Block vs Inline Tags

HTML madhe don prakarche tags astat:

Block Level: He tags navin line pasun suru hotat ani poorn width ghetat. (e.g. <div>, <h1>, <p>)

Inline: He tags fakt garjeitkiach jag ghetat. (e.g. <span>, <a>, <img>)

5. The Metadata Tag (<meta>)

He tags website var disat nahit pan Google Search (SEO) sathi khup mahatvache ahet. He <head> tag chya aat lihile jatat.

<meta name="description" content="Learning HTML tags">
<meta charset="UTF-8">
Quick Note: HTML madhe 100 peksha jast tags ahet, pan daily vaprat fakt 20-30 tags yetat. Tevde shikle tari tumhi professional website banvu shakta!
